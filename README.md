# 🏥 Hospital Management System (MySQL)

## 📌 Project Overview

The Hospital Management System is a relational database project developed using MySQL. It simulates the operations of a hospital by managing departments, doctors, patients, appointments, medicines, and billing information.

The project demonstrates database design, normalization, SQL queries, joins, aggregate functions, and subqueries using a real-world scenario.

---

## 🚀 Features

- Hospital database creation
- Department management
- Doctor management
- Patient management
- Appointment scheduling
- Medicine inventory
- Billing management
- Data retrieval using SQL queries
